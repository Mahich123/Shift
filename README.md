## Data Types

```
Text Type:	str
Numeric Types:	int, float
Sequence Types:	list
Mapping Type:	dict
Boolean Type:	bool
None Type:	NoneType
```

## Get Started

- Create a variable and print it.

```python
msg = "Hello, World!"

print(msg)
```

- Create a function and return any.

```python
def Message():
    return "Hello, World!"
```

- Create a sum and print it.

```python
a = 3
b = 4
c = a + b * 2

print(c)
```
